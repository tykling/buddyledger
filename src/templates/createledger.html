Create new ledger

To create a new ledger you need to come up with a name and choose a native currency for the ledger.

What is native currency ?
  • All amounts throughout the system will always be displayed in the original currency and the ledgers native currency.
  • Expenses can be in any currency, but will always default to the ledgers native currency.
  • Payments for a given expense will always be in the currency of the expense.
  • Backpayments can be in any currency but will also default to the native currency for the ledger.
